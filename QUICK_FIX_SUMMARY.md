# Quick Fix Summary - Published Exams Not Showing

## The Problem
Students can't see published exams on their dashboard.

## The Fix (3 Simple Steps)

### 1. Check the Problem
Go to: `/exam-diagnostic`

This shows which exams are broken.

### 2. Fix It
Go to: `/migrate-exams`

Click "Run Migration Now"

Wait 5-10 seconds.

### 3. Done!
Students refresh their browser and can now see exams.

## That's It!

The migration:
- Fixes all existing exams
- Takes less than 10 seconds
- Safe to run multiple times
- Permanent fix

## New Exams
All exams created from now on work automatically. No migration needed.

## Still Having Issues?

1. Make sure exam is marked as "Published" (not "Draft")
2. Check exam isn't restricted to wrong department
3. Ask student to hard refresh (Ctrl+Shift+R)
